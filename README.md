# Purrmission

Purrmission is a playful purchase-decision advisor. Tell the cat what you want to buy, how much it costs, how often you will use it, how long you expect to keep using it in months or years, and how impulsive the purchase feels. The cat gives a verdict, estimates cost per expected use, lets you negotiate better terms, and reacts if you ignore the advice.

## Run locally

Open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173/`.
